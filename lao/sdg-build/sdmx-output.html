Lao SDG data summary

SDMX output

This output has an SDMX file for each indicator's data, plus one SDMX file with all indicator data. This data uses numbers and codes only, so is not specific to any language.The metadata is output in a 'meta' subfolder' and similarly has one per indicator plus an 'all' file. Translations of the metadata are included in each file using the 'lang' attribute. Examples are below:

  • sdmx/all.xml (data)
  • sdmx/meta/all.xml (metadata)
  • sdmx/1-1-1.xml (data)
  • sdmx/meta/1-1-1.xml (metadata)
  • sdmx/1-2-1.xml (data)
  • sdmx/meta/1-2-1.xml (metadata)
  • etc...
